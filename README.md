# Vacation-House-Rental
How to Run the Project (Using Laragon)
1) Download and install Laragon
2) Move the project folder into:
C:\laragon\www
3) Start Laragon
4) Open your browser (e.g., Google Chrome)
5) Go to:
http://localhost/Vacation-House-Rental-main/
